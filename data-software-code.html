Tutorials

Multi-panel plots with R

Developed for BIOL 548L (Visual and Oral Presentations), graduate course at UBC. Learning objectives include:

  • Determining how to import & wrangle data from diverse sources.

  • Using the grammar of graphics to construct plots in ggplot2.

  • Planning a sequence of plots that support a declarative statement of a result.

  • Constructing multi-panel, publication-quality figures using code.
